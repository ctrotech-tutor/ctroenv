We just released CtroEnv v1.0 — a TypeScript-first environment variable toolkit that I wish existed years ago. 🚀

The idea is simple: define your env schema once, and get type inference, runtime validation, CLI tooling, and auto-generated docs — all from a single definition.

Zero dependencies in the core package. Works with Node.js, Vite, and Next.js out of the box.

If you've ever had a production crash because someone forgot to set DATABASE_URL, you know why this exists.
